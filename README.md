# vim-data-URI-scheme-base64.vim

## Install
Example for vim-plug:

Adding 
```
Plug 'fijixfiji/vim-data-URI-schem-base64.vim'
```
in `.vimrc`.

Then `:PlugInstall`.

### requirements
Vim with Python3 (`if_python3`)


## Usage
`:InputeDataURISchemeBase64 path/to/your/image/file.png`
